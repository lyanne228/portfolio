  1. Use the arrow keys to move around.
  2. Use the W and S keys to pitch up and down, the A and D keys to rotate left and right (yaw), and the Z and X keys to roll left and right.
  3. Use the C key to cruise forward (to experience plane navigation).
  4. Use the Spacebar to make the lamp jump.
  5. Use the R key or the 'Reset Position' button to return to the default position.
  6. Use the F key or the 'Stop/Run' button to return to stop and start the lamp animation. After clicking either button, be sure to click outside the button.
    Failure to do so could result in glitches when making the lamp jump.
  7. Use the buttons/settings fields to change the lighting and shading settings.
